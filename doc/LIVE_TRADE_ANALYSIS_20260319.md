## 八、策略优化建议

### 🔴 高优先级（直接影响盈亏）

#### 建议 1：提高 TWR=0.860 信号的过滤条件

**问题**：赔率强信号（mkt<0.85，TWR=0.860）实际胜率 78%，比理论值低 8%，且是分1/分2亏损的主要来源。

**建议**：
- 分1/分2 的 TWR=0.860 信号，要求 `min_gap >= 0.10%`（当前 0.03%）
- 分1 的 TWR=0.860 信号，可考虑直接禁止入场
- 或将 TWR=0.860 降低至 0.820，重新计算 EV 以提高过滤门槛

#### 建议 2：提高 TWR=0.968（gap大，末期套利）的门槛

**问题**：该档实际胜率 84%，比理论值 96.8% 低 12.8%，PnL 为负（-$3.02）。

**建议**：
- 该类信号要求更晚入场（仅分3/分4）
- 或提高最低 EV 门槛至 0.08（当前 0.05）

#### 建议 3：将 EV 门槛从 0.05 提高到 0.08

**问题**：EV=0.05~0.08 的信号（24 笔）PnL 仅 +$11.90，盈利能力薄弱。

**建议**：`MIN_EV_THRESHOLD=0.08`，减少低效信号，专注高质量机会。

---

### 🟡 中优先级（提升策略质量）

#### 建议 4：强化低赔率机会的投注比例

**发现**：entry_price<0.55 的机会占 186 笔中的 37%（55 笔），却贡献了 83% 的总收益，胜率 96.4%。

**建议**：
- 对赔率<0.55 的信号，Kelly 系数可适当上调（当前受 MAX_BET_FRACTION 限制）
- 建议设置 `HIGH_CONF_BET_FRACTION=0.25`（当前 MAX_BET_FRACTION=0.10），仅用于赔率<0.55 场景

#### 建议 5：分2 UP 信号单独降级处理

**发现**：分2 UP 胜率 65%（最低），而分2 DOWN 胜率 90.5%。

**建议**：分2 UP 信号要求额外条件，如：
- `entry_price < 0.80`（排除高赔率时的 UP 信号）
- `min_gap >= 0.15%`（比当前更强的 gap 要求）

#### 建议 6：Paper 模式添加资金上限

**问题**：Paper 模式资金通过复利从 $30 增长到 $2333，导致后期注额脱离实际（单笔 $112）。

**建议**：在 Paper 模式下，强制 `max_capital_paper = initial_capital * 3`（限制为 $90），避免数据失真。

---

### 🟢 低优先级（长期优化）

#### 建议 7：05时段（凌晨5点）降低入场敏感度

**发现**：05时胜率 69.2%，低于其他时段。

**建议**：05:00~05:59（UTC+8）入场时，要求 `min_conf >= 0.55`（当前 0.40）。

#### 建议 8：添加连败熔断保护

**发现**：最大连败 2 次，当前保护已够用，但实盘 Kelly 复利失控风险需防范。

**建议**：
- 连败 2 次后，暂停 1 个窗口
- 实盘中严格执行 `MAX_BET_FRACTION=0.10`，不超过 $3/笔（资金 $30）

当前 `observations` 表为空，无法分析：
- 各信号路径触发频率
- 被过滤掉的信号（EV 不足、无信号）分布
- MarketContext 置信度的实际分布

**建议**：恢复 `collect_data.py` 中的 observations 写入逻辑，供后续分析。

## 九、总结

### 策略核心价值确认

策略的核心盈利来源非常集中：

```
低赔率机会（entry_price < 0.55）
  ├── 分3/分4 入场
  ├── EV > 0.20
  └── 贡献总 PnL 的 97%
```

这类机会出现时（约占总信号 37%），策略表现极为稳健。

### 主要风险来源

```
早期信号（分1/分2）
  ├── TWR=0.860（赔率强信号）：实际胜率仅 78%
  ├── 分2 UP 信号：胜率 65%
  └── 亏损主要集中在 05~07时、EV=0.08~0.20 区间
```

### 建议优先执行的参数调整

| 参数 | 当前值 | 建议值 | 原因 |
|------|--------|--------|------|
| `MIN_EV_THRESHOLD` | 0.05 | **0.08** | 低 EV 信号盈利能力弱 |
| 分1 TWR=0.860 | 允许 | **禁止** | 胜率 73%，亏损概率高 |
| 分2 UP | 当前门槛 | **price<0.80** | 65% 胜率不达标 |
| 低赔率(<0.55)投注 | MAX×0.10 | **MAX×0.25** | 96.4% 胜率，可适当加仓 |
| Paper 资金上限 | 无限复利 | **初始×3** | 避免数据失真 |
