## 核心问题

### 1. 低波动环境高亏损率
- **数据**: 80%亏损单发生在"波动↓"环境
- **案例**: #319 Up@0.78 gap+0.02% → -$4.95

### 2. 震荡市信号不可靠
- **数据**: 震荡市（可信度<0.55）亏损率高
- **案例**: #326/#327 Down@0.79/0.815 conf=0.48 → 反向结算

### 3. 分3高赔率gap不足
- **数据**: gap<0.10%的高赔率信号易反向
- **案例**: #325 Down@0.785 gap-0.01% 剩余31s → -$3.92

## 优化方案（已实施）

### P0优化（核心）

#### 1. 低波动环境路径2赔率门槛提高
```python
if vol_level == "low" and odds_px < 0.85:
    return None  # 低波动时赔率必须≥0.85
```

#### 2. 震荡市双确认加强
```python
if minute <= 3 and signal_confidence < 0.55:
    if odds_px < 0.85 and not (gap_same_dir and abs(gap) >= 0.10):
        return None  # 震荡市要求：高赔率或强gap
```

#### 3. 分3高赔率gap门槛提高
```python
if minute >= 3 and odds_px > 0.85:
    gap_ok = gap_same_dir and abs(gap) >= 0.10  # 从0.05→0.10
```

### P1优化（增强）

#### 4. 路径2 EV门槛动态调整
```python
if vol_level == "low":
    ev_threshold = base_ev * 2.0  # 低波动EV门槛翻倍
elif _local_hour == 23:
    ev_threshold = base_ev * 1.75
elif signal_confidence < 0.50:
    ev_threshold = base_ev * 1.5
```
